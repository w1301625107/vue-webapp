## 运行

### 注意项目需要node环境

```bash
#进入项目根目录
#安装
npm i
# 启动
npm run dev
#打开浏览器地址栏输入localhost:8080即可
```

##注意
.vue 文件中不要有空的style，不然编译出的文件会多出空的css
